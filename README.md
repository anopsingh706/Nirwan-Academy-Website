# 🎓 Nirwan Academy Website

A modern, responsive website built for **Nirwan Academy** — a higher primary school and coaching institute specializing in preparation for **Jawahar Navodaya Vidyalaya (JNV)** and **Sainik School**.

🌐 Live Demo: https://nirwan-academy-website.vercel.app/  
📁 Repository: https://github.com/anopsingh706/Nirwan-Academy-Website  

---

## 🚀 Features

- 🎯 Modern UI with responsive design (Mobile + Desktop)
- 🖼️ Dynamic Hero Section with image carousel
- 🏆 Student Achievements & Results Showcase
- 📢 Marquee Section for selected students
- 🧑‍🏫 Faculty & Courses Information
- 🏫 Admissions & Enquiry Forms
- 📩 Email integration for enquiry submissions
- 📍 Google Maps integration for location
- 🖼️ Dedicated Gallery Section
- 💬 Floating WhatsApp & Call buttons
- 🧾 Director’s Message Section
- 🎨 Clean and professional design with Tailwind CSS

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/anopsingh706/Nirwan-Academy-Website.git
cd Nirwan-Academy-Website
2. Install dependencies
npm install
3. Run development server
npm run dev
4. Build for production
npm run build
5. Preview production build
npm run preview
📸 Image Handling Guidelines
Place all images inside the public/ folder
Use kebab-case naming (recommended)

✅ Example:

hero-section.webp
student-achievement.webp

❌ Avoid:

Hero Section.webp
image 1.webp
Use paths like:
image: '/hero-section.webp'
🌐 Deployment (Vercel)
Connect GitHub repository to Vercel
Use the following settings:
Build Command: npm run build
Output Directory: dist
Install Command: npm install
Deploy 🚀
